Market Class

A class to calculate market risk for the Solvency Capital Requirement (SCR).

This class is responsible for calculating various market risk components as part of the SCR calculation process. It handles different types of market risks including interest rate risk, equity risk, property risk, spread risk, concentration risk, and currency risk.

The class is split across several modules:

  • market.py: The main class file containing core market risk calculation functions

  • bond.py: Functions for bond valuation and related calculations

  • aggregate.py: Functions for aggregating different market risk components

  • credit.py: Functions for calculating credit risk

  • conc.py: Functions for calculating concentration risk

Limitations

  • Need to resolve the day count issue. Currently the fucntion uses ACTUAL/ACTUAL. In reality it might be better to use ACTUAL/365

  • No allowance is made for the assest shocks. This would need to be a seperate data import, not very complex.’

  • We make no accurate calcaultion of the dirty market price.

  • In the calcualtion we disocunt all cashflows at the risk-free rate. In reality there is a spread on almsot all assets. We could add this but that would require a newton-raphson approximation.
